    准备工作 #

    物业系统开始使用前需要先做一些设置及准备工作。系统迁移，数据迁移等功能。迁移完成后，您将获得以下功能：

    • 住户可以使用app水电充值
    • 住户可以使用app在线缴物业费
    • 住户可以使用app蓝牙开门功能
    • 住户可以使用app在线缴停车费
    • 住户在线投诉保修
    • 住户在线保洁预约

    企业设置 #

    查看视频

    • 配置当前企业的基本信息、收款账户绑定等。视频中有详细说明，请查看视频。
    • 要与longpay支付关联起来，则此处需要配置收款账户。 （否则APP端的支付，无法使用longpay方式）

      物业首次入住只需要设置收款账户信息，绑定龙支付

    费用设置 #

    • 主要用来设置物业费，水电，燃气，车位费价格，计费方式

      基本的费用设置，目前已经设置好，暂时不做详细说明

    用户数据迁移 #

    1. 从旧系统迁移到新的系统，由于用户还未入住，可以选择人工导入用户数据
    2. 用户也可以从APP注册登录，
    3. 后台管理员可以在管理后台关联用户
