    4. 保洁

    保洁 #

    保洁和投诉、报修流程基本类似都是采用工作流的方式执行。需要预先设置保洁跟进人，方便安排工作。

    购买保洁服务包 #

    1. 用户可以根据自己入住的房间在APP端购买服务包
    2. 购买的服务包只限当前房间在有效期内使用
    3. 服务包购买完成后，可以进行保洁预约

    预约保洁 #

    1. 用户的房间如果需要进行保洁，可以在已购买的服务包订单下进行预约。
    2. 预约过程根据需要选择额外服务，填写相关信息后下单
    3. 下单后会自动生成保洁工单，保洁跟进人可以在app或者管理后台进行保洁工单分配
    4. 保洁完成后，用户可以在APP进行评价

    保洁工单 #

    工单执行过程按照工作流方式执行，可以进行分配给其他人或者本人执行操作，工单完成后请联系物业管理人员

    保洁退费 #

    如遇特殊情况，需要取消保洁的情况。

    1. 如果无额外服务，取消订单，自动退回预约次数。
    2. 如有额外服务，取消订单，自动退回预约次数，额外服务的费用退回需要联系管理员。 3.管理员再后台账单模块查看退费订单，根据实际情况进行操作。具体操作请参考账单审核步骤.

    保洁服务包过期 #

    在有效期内未使用完成的服务包会自动过期，不做退费处理
