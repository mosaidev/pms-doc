    水电充值 #

    1. 用户在APP端进行水电充值界面，给指定的房间进行充值
    2. 充值完成后，管理员可以在管理后台看到水电充值审核记录
    3. 管理员在后台查看充值订单，根据订单的金额在原有水电充值的系统进行充值
    4. 用户或管理员线下执行上表充值，充值完成后抄去余量读数
    5. 将余量数据登记到充值记录上表执行中即完成充值

    水电批量导入 #

    查看视频

    1. 从水电导入模块下载导入模板，导入的模板包含了所有的房源信息
    2. 根据下载的模板房间，对应好读数，根据需求按时进行抄表
    3. 抄表完成后需要确认数据是否有误
    4. 将数据导入信息，预览确认数据是否正确
    5. 确认无误后点击确认按钮导入数据
    6. 导入完成后的数据可以在计费列表查看
