    用户入住 #

    用户使用APP后：

    • app水电充值
    • app在线缴物业费
    • app蓝牙开门功能
    • app在线缴停车费

    管理员关联录入 #

    查看视频

    1. 管理员在管理后台找到对应的房源列表页面
    2. 找到指定的房间，查看详情
    3. 在详情中点解check in 按钮添加用户信息

    用户在APP端注册 #

    查看视频

    1. app端注册的用户需要进行审核，审核通过后才可以查看对应的房间

    设置主要联系人 #

    1. 在房源详细页面查看对应的房间用户列表
    2. 指定用户被设置为主要联系人，该用户可以对其他用户进行审核

    用户移除 #

    如果用户已经搬离，需要对用户进行移除

    • 管理员关联录入
    • 用户在APP端注册
    • 设置主要联系人
    • 用户移除
